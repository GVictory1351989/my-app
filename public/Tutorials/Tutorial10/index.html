🌐 Web Storage: localStorage vs sessionStorage

✅ What is Web Storage?

  • localStorage stores data with no expiration date
  • sessionStorage stores data for the current tab session only

📦 Storing Data

// String
localStorage.setItem('firstName', 'Asabeneh');

// Number
localStorage.setItem('age', 200);

// Array
const skills = ['HTML', 'CSS', 'JS', 'React'];
localStorage.setItem('skills', JSON.stringify(skills));

// Object
const user = {
  firstName: 'Asabeneh',
  age: 250,
  skills: ['HTML', 'CSS', 'JS', 'React']
};
localStorage.setItem('user', JSON.stringify(user));

📤 Getting Data

let firstName = localStorage.getItem('firstName');
let age = localStorage.getItem('age');
let skills = JSON.parse(localStorage.getItem('skills'));
let user = JSON.parse(localStorage.getItem('user'));

console.log(firstName, age, skills, user);

❌ Remove Item or Clear All

localStorage.removeItem('age');
localStorage.clear(); // removes everything!

🧪 sessionStorage Example

sessionStorage.setItem('sessionName', 'Temporary Name');
sessionStorage.getItem('sessionName');

🔍 How to View Stored Data in Browser

  1. Open DevTools (Right Click → Inspect)
  2. Go to Application Tab
  3. Check Local Storage and Session Storage
